Exercícios em Java Script



Pilha


Elabore um programa que gera uma pilha de 5 elementos, cujo primeiro elemento é 1 e o último é 5, com o uso de array. O programa deve apresentar a geração da pilha mostrando o resultado gerado passo a passo com o console.log, e depois remova todos os elementos da pilha mostrando passo a passo com o console.log, usando o conceito de LIFO (Last In, First Out).







Fila


Elabore um programa que gera uma fila de 9 elementos, cujo primeiro elemento é 1 e o último é 9, com o uso de array. O programa deve apresentar a geração da fila mostrando o resultado gerado passo a passo com o console.log, e depois remova todos os elementos da fila mostrando passo a passo com o console.log, usando o conceito de FIFO (First In, First Out).







Matriz


Elabore um programa que gera duas matrizes 5 x 5, sendo a primeira matriz com valores iguais a 20 na diagonal principal, valor igual a 15 na diagonal secundária e os elementos que sobrarem com números aleatórios em um intervalo de 1 à 10. Enquanto a segunda matriz tem seus valores iguais a 1 nas linhas de índices pares e valores iguais a 2 nas linhas de índices ímpares. Após gerar as duas matrizes, faça a soma entre elas. Use o laço “for” para elaborar a lógica do programa. Observação: Para a função “random”, use a seguinte expressão: valorAleatorio = Math.floor(Math.random() * (max - min + 1) ) + min

While


É uma estrutura de controle que permite executar um bloco de código repetidamente enquanto uma determinada condição for verdadeira. É uma ferramenta poderosa para automatizar tarefas e iterar sobre conjuntos de dados.





Exercício 1


Crie um programa que percorra um array de 10 números e calcule a soma de todos os números divisíveis por 5, utilizando um laço do...while, considerando o seguinte array: [5, 10, 14, 25, 30, 7, 3, 50, 9, 45].




Exercício 2


Crie um programa que percorra um array de 10 números e calcule a soma de todos os números divisíveis por 3, utilizando um laço while, considerando o seguinte array: [9, 14, 21, 33, 18, 20, 27, 10, 12, 16].
